# Proyecto: Fundamentos de tecnologias web

Mediante el siguiente proyecto se busca dar a conocer los conocimientos aprendidos a lo largo de la experiencia educativa por medio del uso de las prinicipales tecnologias web:

- Html
- CSS
- Js

Asi mismo demostrar el uso de tecnologias de apoyo como archivos markwdown para la documentacion del proyecto, git y github para control de versiones y xml para el guardado de datos.

Para la generacion de el XML e imagenes se uso IA generativa y se enlistan los prompts usados para su generacion:
